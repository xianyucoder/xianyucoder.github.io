解决代码自动上传后运行报错

当你上传代码后，运行后报错“无法找到对应的文件夹/文件”,这是因为这个时候我们的代码仅仅上传到了服务器上，但是我们的Python解释器是运行在容器中，而我们容器查找代码是查找的数据卷，所以这个时候就需要我们在配置中做一个简单的地址映射。
