еҸ‘иЎЁдәҺ 2019-04-04 | жӣҙж–°дәҺ 2019-08-22 | еҲҶзұ»дәҺ Docker | иҜ„и®әж•°пјҡ | ж¬Ўйҳ…иҜ»
жң¬ж–Үеӯ—ж•°пјҡ 3.8k | йҳ…иҜ»ж—¶й•ҝ ≈ 6 еҲҶй’ҹ

д»Җд№ҲжҳҜDockerfile?

Dockerfileе…¶е®һе°ұжҳҜж №жҚ®зү№е®ҡзҡ„иҜӯжі•ж јејҸж’°еҶҷеҮәжқҘзҡ„дёҖдёӘжҷ®йҖҡзҡ„ж–Үжң¬ж–Үд»¶пјҢеҸҜд»ҘдҪҝз”Ёdocker buildе‘Ҫд»Өдҫқж¬Ўжү§иЎҢеңЁDockerfileдёӯе®ҡд№үзҡ„дёҖзі»еҲ—е‘Ҫд»ӨпјҢжңҖз»Ҳз”ҹжҲҗдёҖдёӘж–°зҡ„й•ңеғҸ

еҰӮжһңдҪ жғіиҰҒе®ҡеҲ¶еұһдәҺиҮӘе·ұзҡ„й•ңеғҸDockerfileжҳҜдҪ еҝ…дёҚеҸҜе°‘зҡ„е·Ҙе…·гҖӮ

docker buildзҡ„дҪҝз”Ё

жғіиҰҒжһ„е»әе®ҡеҲ¶еҢ–зҡ„й•ңеғҸпјҢеҸҜд»ҘдҪҝз”Ёdocker buildиҝҗиЎҢDockerfileжһ„е»әй•ңеғҸ

е‘Ҫд»Өж јејҸ: docker build [еҸӮж•°] PATH | URL

е…¶дёӯ:

PATH жҳҜDockerfileжүҖеңЁи·Ҝеҫ„

URL жҳҜDockerfileжүҖеңЁзҡ„URL

еёёз”ЁеҸӮж•°:

е…ідәҺDockerfileжһ„е»әиҝҮзЁӢзҡ„жө…жһҗ

еңЁд№ӢеүҚе…ідәҺDockerй•ңеғҸдёҺе®№еҷЁзҡ„ж–Үз« дёӯпјҢе’ёйұјеҜ№й•ңеғҸзҡ„ж–Үд»¶зі»з»ҹеҒҡдәҶдёҖдёӘз®ҖеҚ•зҡ„и§ЈйҮҠпјҢдёҚзҶҹжӮүзҡ„жңӢеҸӢеҸҜд»ҘеҶҚжё©д№ дёҖдёӢгҖӮ

жҲ‘д»¬е…ҲзңӢдёӢеҸҜд»ҘзӣҙжҺҘдҪҝз”Ёзҡ„Dockerfileз»“жһ„жҳҜд»Җд№Ҳж ·зҡ„:

зҺ°еңЁз®ҖеҚ•и®Іи®Іе’ёйұјеҜ№Dockerfileжһ„е»әиҝҮзЁӢзҡ„зҗҶи§Јпјҡ

Dockerfileзҡ„жү§иЎҢйЎәеәҸжҳҜиҮӘдёҠиҖҢдёӢиҝӣиЎҢзҡ„пјҢеҪ“з¬¬дёҖж¬ЎдҪҝз”ЁDockerfileжһ„е»әй•ңеғҸж—¶Dockerfileдёӯзҡ„е…ЁйғЁе‘Ҫд»ӨйғҪдјҡжү§иЎҢдёҖиҫ№жңҖеҗҺжһ„е»әдёәдёҖдёӘе®Ңж•ҙзҡ„й•ңеғҸгҖӮ

еңЁжһ„е»әзҡ„иҝҮзЁӢдёӯпјҢжҜҸжү§иЎҢдёҖжқЎе‘Ҫд»ӨDockerйғҪдјҡжү§иЎҢдёҖж¬Ўcommitе‘Ҫд»ӨпјҢжҺҘдёӢжқҘзҡ„жҜҸдёҖжқЎе‘Ҫд»ӨйғҪжҳҜеңЁеүҚйқўжүҖжңүе‘Ҫд»ӨйӣҶжҲҗзҡ„й•ңеғҸеҹәзЎҖдёҠеҶҚж¬Ўжү§иЎҢзҡ„гҖӮ

иҝҷж ·зҡ„жү§иЎҢжңүд»Җд№ҲеҘҪеӨ„е‘ўпјҢеҒҮи®ҫеҪ“дҪ жү§иЎҢиҝҮдёҖж¬Ўзҡ„й•ңеғҸжһ„е»әеҗҺжғіиҰҒеңЁDockerfileдёӯй—ҙйғЁеҲҶж·»еҠ дёҖиЎҢDockerfileзҡ„е‘Ҫд»ӨпјҢйӮЈд№ҲеңЁжү§иЎҢж–°зҡ„Dockerfileе‘Ҫд»Өж—¶пјҢDockerеҸӘдјҡжү§иЎҢж–°еўһе‘Ҫд»Ө(еҢ…жӢ¬ж–°еўһе‘Ҫд»Ө)д№ӢеҗҺзҡ„жһ„е»әе‘Ҫд»ӨпјҢеӨ§еӨ§еҮҸе°‘дәҶжҖ§иғҪзҡ„жҚҹиҖ—гҖӮ

Dockerfileеёёз”Ёе‘Ҫд»Өи§Јжһҗ

жҺҘдёӢжқҘе°ұжҳҜе…ідәҺDockerfileе‘Ҫд»Өзҡ„д»Ӣз»ҚжӣҙеӨҡе®һдҫӢеҸҜд»ҘеҸӮиҖғе®ҳж–№зҡ„Dockerfileе®һдҫӢ:https://github.com/docker-library/docs

е…ідәҺDockerfileе‘Ҫд»Өзҡ„д»Ӣз»Қд№ҹеҸҜд»ҘеҸӮиҖғ:https://docs.docker.com/engine/reference/builder/#usage

RUN

жһ„е»әй•ңеғҸиҝҮзЁӢдёӯйңҖиҰҒжү§иЎҢзҡ„е‘Ҫд»ӨпјҢеҸҜд»Ҙжү§иЎҢеӨҡжқЎ

еҗҢж—¶RUNе‘Ҫд»ӨеҗҺеҸҜд»ҘдҪҝз”ЁдёӨз§ҚеҪўејҸиҫ“е…ҘиҰҒжү§иЎҢзҡ„е‘Ҫд»Ө execдёҺshell

еҪ“дҪҝз”Ёexecж–№ејҸиҫ“е…Ҙжү§иЎҢзҡ„е‘Ҫд»Өж—¶пјҢе‘Ҫд»Өж јејҸжҳҜjsonж јејҸзҡ„пјҢе‘Ҫд»ӨеңЁеҪ“еүҚиҝӣзЁӢжү§иЎҢ

еҪ“дҪҝз”Ёshellж–№ејҸиҫ“е…Ҙжү§иЎҢзҡ„е‘Ҫд»Өж—¶пјҢе‘Ҫд»ӨеңЁеӯҗиҝӣзЁӢдёӯжү§иЎҢ(жҺЁиҚҗдҪҝз”Ёshellж–№ејҸ)

CMDдёҺENTRYPOINT

CMDжҳҜж·»еҠ еҗҜеҠЁе®№еҷЁжҳҜйңҖиҰҒжү§иЎҢзҡ„е‘Ҫд»ӨпјҢеӨҡжқЎе‘Ҫд»ӨеҸӘжңүжңҖеҗҺдёҖжқЎз”ҹж•ҲпјҢеҸҜд»ҘеңЁеҗҜеҠЁе®№еҷЁдәӢиў«иҰҶзӣ–е’Ңдҝ®ж”№гҖӮ
CMDе‘Ҫд»Өж јејҸжңүдёүз§ҚпјҢexecгҖҒshellгҖҒй»ҳи®ӨжҸҗдҫӣз»ҷENTRYPOINTзҡ„е‘Ҫд»Ө

  • е‘Ҫд»Өж јејҸдёәshellеҪўејҸпјҢе‘Ҫд»ӨжҳҜд»ҘеӯҗиҝӣзЁӢзҡ„еҪўејҸжү§иЎҢ
  • е‘Ҫд»Өж јејҸдёәjsonеҪўејҸпјҢе‘Ҫд»ӨеңЁеҪ“еүҚиҝӣзЁӢжү§иЎҢ(жҺЁиҚҗдҪҝз”Ё)
  • е‘Ҫд»Өж јејҸдёәjsonеҪўејҸпјҢдё”жүҖжңүjsonеҶ…е®№дёәеҸӮж•°еҪўејҸпјҢеҲҷй»ҳи®ӨжҸҗдҫӣз»ҷENTRYPOINT

ENTRYPOINTзҡ„дҪҝз”ЁCMDдёҺзӣёеҗҢпјҢдҪҶиҝҷдёӘдёҖе®ҡдјҡиў«жү§иЎҢпјҢдё”дёҚдјҡиў«иҰҶзӣ–е’Ңдҝ®ж”№

LABELдёҺMAINTAINER

LABELз”ЁдәҺдёәй•ңеғҸж·»еҠ еҜ№еә”зҡ„ж•°жҚ®

ж·»еҠ зҡ„ж•°жҚ®ж јејҸдёә:LABEL key=value вҖҰ. Key=value

MAINTAINER:з”ЁдәҺиЎЁзӨәй•ңеғҸзҡ„дҪңиҖ…(еҚіе°Ҷиў«йҒ—ејғдҪҝз”Ё)

ENVдёҺARG

ENVз”ЁдәҺи®ҫзҪ®жү§иЎҢе‘Ҫд»Өж—¶зҡ„зҺҜеўғеҸҳйҮҸпјҢе№¶дё”еңЁжһ„е»әе®ҢжҲҗеҗҺпјҢд»Қз„¶з”ҹж•Ҳ

ARGз”ЁдәҺи®ҫзҪ®еҸӘеңЁжһ„е»әиҝҮзЁӢдёӯдҪҝз”Ёзҡ„зҺҜеўғеҸҳйҮҸпјҢжһ„е»әе®ҢжҲҗеҗҺж¶ҲеӨұ

ADDдёҺCOPY

ADDз”ЁдәҺе°Ҷжң¬ең°ж–Үд»¶жҲ–зӣ®еҪ•жӢ·иҙқеҲ°й•ңеғҸзҡ„ж–Үд»¶зі»з»ҹдёӯпјҢдё”иғҪи§ЈеҺӢзү№е®ҡж јејҸж–Үд»¶пјҢдё”иғҪе°ҶURLдҪңдёәиҰҒжӢ·иҙқзҡ„ж–Үд»¶(дјҡе…Ҳе°ҶURLзҡ„ж–Үд»¶е…ҲдёӢиҪҪдёӢжқҘеҶҚжӢ·иҙқ)

COPYе°Ҷжң¬ең°ж–Үд»¶жҲ–зӣ®еҪ•жӢ·иҙқеҲ°й•ңеғҸзҡ„ж–Үд»¶зі»з»ҹдёӯ(жҺЁиҚҗдҪҝз”ЁCOPY+RUN)пјҢеӣ дёәADDе‘Ҫд»ӨеҜ№дәҺйңҖиҰҒи§ЈеҺӢзҡ„ж–Үд»¶ж”ҜжҢҒзҡ„дёҚжҳҜйқһеёёжҷәиғҪ
