在linux系统下使用split可以分割文件,对于多行文件可以使用按行分割的方式,对于单行的大文件可以采用按文件大小分割。

按文件行数分割:split -l 300 large_file.log smallfile_prefix

按文件大小分割:split -b 10m large_file.log smallfile_prefix

之后再按文件读取即可。
